# TIL-Today-I-Learned-
> git 사용법

## git init
#### 리포지토리 초기화. '.git'폴더 생성. 이 폴더에 현재 폴더와 관련된 관리 정보가 저장. working tree라고 부른다. 변경 내역 등 관리.

## git status
#### 리포지토리 상태 표시. 

## git add 파일명
#### 스테이지 영역(커밋 전 임시 영역)에 파일 추가. 
#### git add . : 전체 파일에 추가하는 기능, 요즘 많이 사용! 엄청 편리함. 

## git commit -m '첫 커밋'
#### 실제 커밋에 반영
#### ''로 표시하는 경우 error 발생하여 ""로 표시
